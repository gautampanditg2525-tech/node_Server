Mini User API
Simple Node.js + Express backend demonstrating routing, middleware, and basic authentication.

Install
npm install
Run
npm start
Routes
GET / returns server status
GET /users returns all users
POST /users adds a user
GET /users/:id returns one user
DELETE /users/:id deletes a user
POST /login checks hardcoded credentials
